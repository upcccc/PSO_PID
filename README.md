# PSO_PID

基于粒子群优化（PSO）算法的 PID 参数自整定

## 项目简介

本项目使用**粒子群优化（Particle Swarm Optimization, PSO）算法**对 PID 控制器的参数进行自动整定，以 ITAE（时间乘绝对误差积分）为适应度函数，通过迭代优化找到使控制系统性能最优的 PID 参数组合。

## 项目结构

```
PSO_PID/
├── main.m        # 主程序：PSO 算法主循环，完成参数初始化、迭代优化与结果输出
├── PSO_PID.m     # 适应度函数：调用 Simulink 模型仿真，返回 ITAE 指标
├── PID.slx       # Simulink 控制系统模型：PID 控制器与被控对象
└── README.md     # 项目说明文档
```

## 功能说明

### `main.m` — PSO 主程序

- **初始化**：随机生成 10 个粒子，每个粒子包含 4 个维度（Kp、Ti、Td、Ts），位置和速度初始值均在 `[0, 20]` 范围内。
- **适应度评估**：对每个粒子调用 `PSO_PID` 函数，计算当前参数组合的 ITAE 值。
- **个体最优更新（pbest）**：若当前粒子的适应度优于历史最优，则更新个体最优位置。
- **全局最优更新（gbest）**：记录所有粒子中适应度最小的位置作为全局最优。
- **速度与位置更新**：按照标准 PSO 公式更新粒子速度和位置，并对越界值进行截断处理（速度范围 `[-20, 20]`，位置范围 `[0.01, 50]`）。
- **迭代输出**：共迭代 100 次，每次打印当前迭代次数和全局最优 ITAE 值。
- **结果展示**：输出最优参数组合、最小 ITAE 值及求解耗时，并绘制 PSO 迭代收敛曲线。

### `PSO_PID.m` — 适应度函数

- 接收一个粒子的参数向量 `x = [Kp, Ti, Td, Ts]`，将其赋值给工作区变量。
- 调用 Simulink 模型 `PID.slx` 进行 0～20 秒的仿真。
- 返回仿真结束时刻的输出值作为适应度（即 ITAE 指标）。

### `PID.slx` — Simulink 模型

- 实现了一个完整的 PID 闭环控制系统仿真模型。
- 模型参数（Kp、Ti、Td、Ts）由 PSO 算法在每次迭代中动态调整。

## 优化目标

最小化 **ITAE（Integral of Time-weighted Absolute Error）**：

$$
ITAE = \int_0^T t \cdot |e(t)| \, dt
$$

ITAE 越小，表示系统响应速度越快、超调越小，控制性能越好。

## 算法参数

| 参数 | 值 | 说明 |
|------|----|------|
| 粒子数 | 10 | 种群规模 |
| 迭代次数 | 100 | 最大迭代轮数 |
| 学习因子 c1 | 2 | 个体认知系数 |
| 学习因子 c2 | 2 | 社会认知系数 |
| 惯性权重 w | 1 | 速度继承系数 |
| 速度范围 | [-20, 20] | 粒子最大速度限制 |
| 位置范围 | [0.01, 50] | PID 参数搜索空间 |

## 优化变量

| 变量 | 含义 |
|------|------|
| Kp | 比例系数 |
| Ti | 积分时间常数 |
| Td | 微分时间常数 |
| Ts | 采样时间 |

## 运行环境

- MATLAB（建议 R2018b 及以上版本）
- Simulink

## 使用方法

1. 在 MATLAB 中打开项目目录。
2. 确保 `PID.slx`、`PSO_PID.m` 和 `main.m` 在同一目录下。
3. 在命令行窗口运行：
   ```matlab
   main
   ```
4. 程序将自动完成 PSO 迭代优化，并在结束后输出最优 PID 参数及收敛曲线图。

## 输出示例

```
迭代次数 #1  ITAE= 12.3456
迭代次数 #2  ITAE= 10.8791
...
迭代次数 #100  ITAE= 3.2145
x最佳取值  [8.52, 12.34, 0.76, 0.05]
最小值  3.2145
求解耗时  45.23
```

同时会弹出迭代收敛曲线图，横轴为迭代次数，纵轴为群体最小 ITAE 值。
